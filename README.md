# Mindustry-CustomUnit

一个允许玩家创建自定义单位的Mindustry模组。

## 功能介绍

该模组允许玩家创建、编辑和生产自定义单位，支持调整单位的各种属性，包括：

- 基本属性：生命值、速度、装甲等
- 外观：颜色、尺寸、动画等
- 武器和能力
- 生产配方和时间

## 安装说明

1. 下载最新版本的模组jar文件
2. 将jar文件放入Mindustry的mods目录
3. 启动游戏，在mod列表中启用该模组

## 开发说明

### 构建模组

```bash
./gradlew jar
```

### 开发环境设置

1. 确保安装了Java 17或更高版本
2. 克隆该仓库
3. 在IDE中导入项目
4. 开始开发
